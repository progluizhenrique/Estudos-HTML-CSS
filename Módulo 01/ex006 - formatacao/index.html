Formatando Texto em HTML5

Negrito

A tag reponsável pelo negrito é a <strong>

Itálico

A tag reponsável pelo itálico é a <em>

Texto Marcado

A tag reponsável pela marcação é a <mark>

Texto Excluído

A tag reponsável por excluir um texto é a <del>

Sublinhado

A tag reponsável por sublinhar é a <ins>

Texto Sobrescrito

A tag reponsável por sobrescrever é a <sup>

Texto Subscrito

A tag reponsável por subscrever é a <sub>
