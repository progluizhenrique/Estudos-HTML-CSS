Código fonte e Pré-formatação

Quando for escrever apenas uma linha de código apenas a tag code da conta de representar esse código:

printf("Luiz é lindo");

Já quando é necessário manter a indentação pode se usar a tag pre define o valor como pré-formatado


System.out.println("Digite 10 numeros:");
for (int i = 0; i < 12; i++) {
    lista.add(sc.nextInt());
}

Citações

Para marcar uma citação utilizamos a tag q, exemplo: I will take what is mine with fire and blood

Já para se fazer uma citação completa utiliza a tag blockquote

Numa sala estão sentados três grandes homens, um rei, um sacerdote e um homem rico com o seu ouro. Entre eles está um mercenário, um homem pequeno, de nascimento comum e sem grande inteligência. Cada um dos grandes pede a ele para matar os outros dois. "Faça isso", diz o rei, "pois eu sou seu governante por direito". "Faça isso", diz o sacerdote, "pois estou ordenando em nome dos deuses". "Faça isso", diz o rico,"e todo este ouro será seu". Agora, diga-me: Quem sobrevive e quem morre?

O parâmetro cite é útil para os algoritmos de busca
